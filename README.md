GENDIFF
========

A JS package which shows difference between two files (json).

## Installation
```console
$ git clone git@github.com:Valentina-Vasileva/frontend-project-lvl2.git
$ cd frontend-project-lvl2
$ npm ci
$ npm link
```

Usage
-----
    $ gendiff [--format <fmt>] <path_to_file1> <path_to_file2>

CLI description:

    $ gendiff -h
    $ gendiff --help
